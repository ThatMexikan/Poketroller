Poketroller 1.3
================

Use your Xbox Remote to play PokeMMO
************************************

Right now, the program only works while the key bindings are set to the default values in the game.

XBox Remote       Game

A                 A
B                 B
X                 F1
Y                 F2
D-Pad             Arrow-Keys
Left Bumper       Left
Right Bumper      Right
